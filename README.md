# MTix360 — Mobile Ticketing & Event Management App

A cross-platform mobile application built with **React Native** for seamless event ticketing and management.

---

## ✨ Features

- **Cross-Platform** — Runs on both Android and iOS from a single codebase
- **Stack Navigation** — Smooth screen transitions with React Navigation
- **Date Picker** — Intuitive date selection for event scheduling
- **Searchable Dropdowns** — Easy filtering and selection of events/categories
- **Responsive Layout** — Adapts to any screen size using responsive dimensions
- **Material Design** — Modern UI with gradient backgrounds and dialog popups

---

## 🚀 Getting Started

### Prerequisites

- Node.js 14+
- React Native CLI
- Android Studio (for Android) or Xcode (for iOS)

### Installation

```bash
# Clone the repository
git clone https://github.com/Kushwah1/mtix360_app-master.git
cd mtix360_app-master

# Install dependencies
npm install

# Start Metro bundler
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios
```
